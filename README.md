# Virtual-Coffee-Machine-python
A coin-operated virtual coffee machine simulation in python that processes monetary transactions, tracks ingredient resources, and serves espresso, lattes, or cappuccinos based on user inputs.

Welcome to the Virtual Coffee Machine! This program simulates a real-world vending machine that manages resources (Water, Milk, Coffee), handles coin-based financial transactions, calculates change, and serves classic coffee drinks based on user requests.

## 🎮 Features & Mechanics
- **Menu Management:** Supports three drink options with distinct ingredient requirements and pricing: Espresso, Latte, and Cappuccino.
- **Resource Tracking:** Monitors available levels of water, milk, and coffee beans in real-time, safely rejecting orders if resources are insufficient.
- **Coin Processing:** Accepts virtual quarters, dimes, nickels, and pennies, automatically processing transactions and calculating exact change.
- **Secret Admin Commands:** Features hidden operational controls like `"report"` to print current machine ingredients/earnings and `"off"` to shut down the machine safely.

## ⚙️ Installation & Setup

To run this simulation on your local machine, make sure you have **Python 3** installed.

1. Clone or download this repository to your computer.
2. Ensure your game files (e.g., `main.py`) are placed in the same folder.

## 🚀 How to Run the Game

1. Open your terminal or command prompt.
2. Navigate to the folder where you saved the files.
3. Run the following command:

```bash
python main.py
```
